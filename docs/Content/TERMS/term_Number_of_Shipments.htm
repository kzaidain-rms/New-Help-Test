Number of Shipments

The number of shipments attribute captures the number of relevant cargo shipments in a year. Together with dwell time, the number of shipments attribute considers the transient nature of marine risks and enables accurate loss estimation. The valid range for input is 0-9999. The default is 1 because the model assumes at least 1 shipment is at the geocoded location. This attribute is applicable only for modeling marine cargo and specie exposures.
